# Belajar Java

Repo ini berisi kumpulan file latihan Java dari **Belajar01** sampai **Belajar44**.  
Materi yang dibahas meliputi:
- Percabangan (if, switch)
- Perulangan (for, while, do-while)
- OOP dasar (class, object, method)

## Cara menjalankan
1. Clone repo ini
2. Compile file dengan `javac Belajar01.java`
3. Jalankan dengan `java Belajar01`

## Tujuan
Repo ini dibuat untuk dokumentasi belajar Java step-by-step.
